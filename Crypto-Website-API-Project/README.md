## Exploring the Cryptocurrency Market: A Data-Driven Approach
This project aims to explore the dynamic and volatile world of cryptocurrencies using data analytics. By leveraging APIs, we can gather real-time data on various cryptocurrencies and derive insights that could potentially guide investment strategies.

## Date Source

The data for this project is obtained in real-time from a API provided by [CoinMarketCap](https://coinmarketcap.com/). The API provides detailed information about various cryptocurrencies including their current price, market cap, volume, etc. The data is cleaned and transformed for analysis.
